使用tips：

  • 主要的命令
    hexo init xxx
    hexo new “My New Post”
    hexo server
    hexo generate
    hexo clean && hexo deploy

  • windows上需要使用管理员权限的cmd才可以跑hexo的命令

    如何快捷打开管理员的cmd？
    开始菜单搜索cmd，ctrl_shift_enter打开

  • 如何更换主题
    下载主题到博客的theme中，并在yml文件中设置theme
    常用主题：Hingle、Hacker、 next的地址git clone https://github.com/theme-next/hexo-theme-next.git next

  • 如何设置文章分类（categories/tags 相同处理）

    1. 新建一个页面，命名为 categories 。命令如下：

    2. 编辑刚新建的页面，将页面的类型设置为 categories ，主题将自动为这个页面显示所有分类。

    3. 在当前theme的_config.yml中添加categories的按钮。

  • 设置摘要（两种方式）

    1. description 在Front-matter提供文章摘录，在首页会优先显示这部分内容

    2. 使用<!--more-->在正文标注结束。
